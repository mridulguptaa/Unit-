UnitPlusPlus

UnitPlusPlus a functional testing library for the C++ programming language.


Requirements?
MINGW / gcc compiler with all basic header files.


How to get UnitPlusPlus?
Clone the repository https://github.com/mridulguptaa/UnitPlusPlus.git


How to Use UnitPlusPlus?
1.Add the header file test.h in the same folder of your program. 
2.Save expected output in a .txt file saved in the same folder. Each testcase shall be in a separete line starting with "#TestCase" (without quotes)
3.Compile your program. Sample program file has been provided for reference.
